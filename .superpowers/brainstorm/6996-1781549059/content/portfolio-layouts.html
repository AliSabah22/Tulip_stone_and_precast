Portfolio Section — Layout Options

Which grid approach feels right for a premium stone fabrication brand?

Option A
Masonry / Variable Heights
Cards at different heights — tall, square, and wide tiles. Creates a magazine-style collage that feels editorial and dynamic. Popular on high-end portfolio sites.
Option B
Editorial — Hero + Side Stack
One large featured project dominates, with a stack of smaller thumbnails beside it. Repeated in rows. Gives each hero project a moment to shine.
Option C
Asymmetric Feature Grid
A structured but non-uniform layout — one large anchor tile with smaller tiles around it. More deliberate than masonry, feels architectural and considered.
